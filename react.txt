Installation
************
step-1: Node.js Installation
step-2: npm install -g npm
step-3: npx create-react-app app-name
step-4: npm start

Tailwind CSS integration
************************
step-1: npm install -D tailwindcss
step-2: npx tailwindcss init
step-3: content: [
            "./src/**/*.{js,jsx,ts,tsx}",
        ],
stpe-4: Adding In index.css
        @tailwind base;
        @tailwind components;
        @tailwind utilities;
